2.Tscab_Payroll_DWH_Allowance_Fact

9.Deduction_Fact_UpdateQueries


Note: Data in the Fact Tables will be inserted through service with a code: "Pr_DWH_Fact"

Process of "Pr_DWH_Fact" execution

1. First Fact Tables will be truncated and created.
2. Inserts Data into Fact Tables Monthwise to current Month.
3. Updates the records in AllowanceFact Table values with 0 instead of NULL.
4. Updates the records in DeductionFact Table values with 0 instead of NULL.
